# Jot

Jot is a very basic document editor built with [Sinatra](http://www.sinatrarb.com/) and [Orchestrate](https://orchestrate.io/).

## Installation

1. Fork this repo.
2. `git clone https://github.com/YOUR_NAME/jot.git`
3. `cd jot`
4. `bundle install`
5. `rackup`
6. visit `localhost:9292` in your browser
